【快速FAQ】

Q: 如何在论文中引用这些结果？
A: 建议格式：
   "根据控制流量侧信道特征的IsolationForest检测结果（精确率100%，
    召回率15.0%），作为第一层轻量级过滤器可有效捕获非法源注入攻击..."
   具体数据均可从 detection_results.json 或表3.1-3.2 提取

Q: 这些数字是实时生成的还是固定值？
A: 完全实时生成。运行 side_channel_test.py 后会按相同合成数据
   每次生成相同结果（因为使用随机种子42）。可通过修改数据参数
   或真实PCAP测试得到不同结果。

Q: 如何改进召回率15%？
A: 建议方向已在3.2.2.5节"局限与未来工作"中详述：
   1. 增强IAT特征（引入高阶统计）
   2. 添加TCP标志位/协议字段
   3. 集成多模型（隔离森林+随机森林）
   4. 结合第二层ET-BERT进行深层分析

Q: 能否处理加密流量？
A: 可以。特征工程仅依赖报文头（IP/TCP头），而非载荷。
   内容加密对检测无影响。参考3.2.2.5节"局限"讨论。
