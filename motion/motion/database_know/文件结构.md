# 二、每个文件到底存什么（重点）

---

# 1.action_map.json

这是：

# 动作编号映射表

作用：

* 给所有动作统一编号
* 模型训练必须用
* Transformer Token 必须用
* HMM/Markov 必须用

---

内容：

```json id="0ktpjl"
{
  "0": "lie_down",
  "1": "stand",
  "2": "walk_slow",
  "3": "walk_mid",
  "4": "run_fast",
  "5": "backward",
  "6": "side_jump",
  "7": "forward_jump",
  "8": "backflip",
  "9": "moonwalk",
  "10": "dance2",
  "11": "hello",
  "12": "twist_body"
}
```

---

为什么单独存？

因为：

后续：

* 新增动作
* 修改动作名
* 模型训练

都会依赖它。

它相当于：

# 字典表

---

# 三、normal_sequences.json

这是最重要的数据之一。

它保存：

# 所有合法动作序列

---

例如：

```json id="yo26lu"
[
  {
    "scene": "basic_patrol",
    "sequence": [1,2,3,2,1]
  },

  {
    "scene": "dance_show",
    "sequence": [1,9,10,12,1]
  }
]
```

---

为什么这样设计？

因为：

你真正学习的是：

```text id="l3a6o2"
动作如何演化
```

而不是：

```text id="4ov1rr"
动作本身
```

---

这个文件用于：

| 用途            | 是否核心 |
| ------------- | ---- |
| Markov学习      | ✓    |
| HMM训练         | ✓    |
| LSTM训练        | ✓    |
| Transformer训练 | ✓    |
| 序列预测          | ✓    |

---

# 四、abnormal_sequences.json

保存：

# 人工构造的攻击序列

---

例如：

```json id="2e3ytl"
[
  {
    "type": "illegal_transition",
    "sequence": [4,0],
    "risk": "high"
  },

  {
    "type": "continuous_backflip",
    "sequence": [8,8,8],
    "risk": "critical"
  }
]
```

---

这个文件：

# 非常重要

因为：

比赛时：

你必须：

```text id="sbn2a6"
证明系统能检测异常
```

否则：

只有正常数据没意义。

---

# 六、transition_matrix.json

这是：

# 状态转移概率矩阵

核心中的核心。

---

例如：

```json id="e8z5gh"
{
  "1": {
    "2": 0.28,
    "11": 0.10,
    "9": 0.08
  }
}
```

意思：

```text id="c30g7y"
当前状态=stand

下一动作：
walk_slow 28%
hello 10%
moonwalk 8%
```

---

这是：

# AMNAR真正学习的行为规律

---

* probabilities
* normal_sequences

全部。

# 第四步

```text id="vmk95z"

```

---

# 第五步

# 十二、检测到底怎么运行

例如：

输入：

```text id="3sh0iw"
[1,2,3,4]
```

即：

```text id="i5u6ez"
stand
→ slow
→ mid
→ fast
```

系统：

```text id="hgd73j"
NORMAL
```

---

但：

```text id="q4t0pv"
[4,0]
```

即：

```text id="c3uqyq"
run_fast
→ lie_down
```

查矩阵：

```text id="2dljy2"
P(0|4)=0.0001
```

于是：

```text id="ma0zdx"
risk=0.9999
```

系统报警。
